# 🔔 浏览器通知功能使用指南

## 功能概述

本工具新增了完整的浏览器通知功能，可以在各种操作完成时向用户发送桌面通知，提升用户体验。

## 主要特性

### 1. 🎨 精美的Logo和图标
- **B站风格Logo**: 在导航栏和通知中使用粉色B站风格Logo
- **分类图标**: 不同类型的通知使用不同颜色的图标
  - ✅ 成功：绿色勾选图标
  - ❌ 错误：红色叉号图标  
  - ⚠️ 警告：黄色感叹号图标
  - ℹ️ 信息：蓝色信息图标
  - 🔄 进度：紫色进度图标

### 2. 🔧 智能通知系统
- **自动权限请求**: 页面加载时自动请求通知权限
- **统一管理**: 所有通知通过统一系统管理
- **用户设置**: 可自定义通知行为和显示

### 3. 📋 通知类型

#### 🎯 操作通知
- **同步开始**: 关注列表同步开始时通知
- **同步完成**: 同步成功完成时通知
- **同步失败**: 同步出错时通知
- **一键更新**: 一键更新各阶段通知

#### 📊 状态通知
- **连接状态**: API连接状态变化通知
- **进度更新**: 长时间操作的进度通知
- **错误提醒**: 重要错误信息通知

#### 💾 数据通知
- **数据同步**: 用户统计数据同步通知
- **等级修复**: 等级信息修复通知
- **分组更新**: 分组信息更新通知

## 使用方法

### 1. 📱 通知设置面板

点击导航栏右上角的 🔔 铃铛图标打开通知设置面板：

#### 基本设置
- **启用浏览器通知**: 总开关，控制是否发送通知
- **通知类型选择**: 可分别开启/关闭不同类型的通知
  - 成功通知
  - 错误通知
  - 警告通知
  - 信息通知

#### 高级设置
- **自动关闭时间**: 调整通知自动消失时间（2-10秒）
- **重要通知**: 错误通知可设置为需要用户手动关闭

### 2. 🧪 测试功能
- 点击"测试通知"按钮可预览所有类型的通知效果
- 按顺序发送：成功→信息→警告→错误通知

### 3. 🔄 重置设置
- 点击"重置"按钮可恢复默认设置
- 所有自定义配置将被清除

## 💡 最佳实践

### 1. 权限管理
- 首次使用时允许浏览器通知权限
- 如需重新申请权限，请在浏览器设置中管理

### 2. 通知优化
- 建议关闭信息通知，避免过于频繁
- 错误和警告通知建议保持开启
- 根据使用习惯调整自动关闭时间

### 3. 场景应用
- **日常使用**: 开启成功、错误、警告通知
- **批量操作**: 开启所有通知，便于监控进度
- **后台运行**: 确保通知开启，及时了解操作状态

## 🔧 技术实现

### 通知API
```javascript
// 基础通知
window.BilibiliTool.sendNotification(title, body, options);

// 类型化通知
window.BilibiliTool.sendSuccessNotification(title, body);
window.BilibiliTool.sendErrorNotification(title, body);
window.BilibiliTool.sendWarningNotification(title, body);
window.BilibiliTool.sendInfoNotification(title, body);

// 智能通知（支持用户设置）
window.BilibiliTool.sendNotificationWithSettings(title, body, type);
```

### 设置管理
```javascript
// 获取设置
const settings = window.BilibiliTool.NotificationSettings.getAll();

// 修改设置
window.BilibiliTool.NotificationSettings.set('enabled', true);

// 重置设置
window.BilibiliTool.NotificationSettings.reset();
```

## ❓ 常见问题

### Q: 为什么没有收到通知？
A: 请检查以下项目：
1. 浏览器是否允许通知权限
2. 通知设置中对应类型是否开启
3. 浏览器是否在静音或专注模式

### Q: 通知图标显示异常？
A: 确保网络连接正常，图标为SVG格式，支持现代浏览器

### Q: 如何批量关闭通知？
A: 在通知设置面板中关闭"启用浏览器通知"总开关
