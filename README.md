# 哔哩哔哩关注列表批量管理工具

一个功能强大的哔哩哔哩关注列表管理工具，支持批量操作、数据可视化和智能分类。

## 🎯 项目概述

本工具旨在帮助B站用户更好地管理关注列表，提供智能分析、批量操作、数据可视化等功能，让关注列表管理变得更加高效。

## ✅ 已完成功能

### 🔍 关注列表管理
- ✅ **自动遍历关注列表**: 完整获取所有关注用户数据
- ✅ **数据同步**: 支持增量同步和强制刷新
- ✅ **本地数据存储**: SQLite数据库存储，支持离线查看
- ✅ **搜索筛选**: 支持按用户名、签名内容搜索
- ✅ **多种排序**: 按关注时间、用户名、等级等排序

### 🏷️ 智能分类系统
- ✅ **自动分类**: 基于用户名和签名的智能分类算法
- ✅ **10大分类**: 游戏、科技、知识、生活、娱乐、美食、时尚、汽车、财经、体育
- ✅ **手动调整**: 支持手动修改用户分类

## 安装与使用

### 环境要求

- Python 3.8 或更高版本
- Chrome/Edge 浏览器 (推荐)

### 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/Bilibili-batch-operation-tool.git
   cd Bilibili-batch-operation-tool
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **配置Cookie**
   - 登录哔哩哔哩网页版
   - 获取Cookie信息
   - 在 `config/config.json` 中配置

4. **启动应用**
   ```bash
   # Windows
   start.bat
   
   # Linux/macOS
   ./start.sh
   ```

5. **访问界面**
   打开浏览器访问: http://localhost:8080

#### 智能洞察原则
智能洞察功能基于以下数据和算法为用户提供深度分析：

**分析数据源**
- 用户基础信息（用户名、签名、等级、认证状态）
- 关注时间和关注行为模式
- 分类分布和内容偏好
- 活跃度指标和互动频率

**洞察算法**
- **内容偏好分析**: 根据关注UP主的分类分布，分析用户兴趣偏好
- **关注质量评估**: 基于UP主的粉丝数、认证状态、活跃度等评估关注质量
- **时间模式识别**: 分析关注时间规律，识别用户行为习惯
- **异常检测**: 识别可能的僵尸账号、营销账号等异常关注

**忽略未分组设置**
- 启用后，统计和分析将排除未分组用户（包括"未分组"、"默认分组"、"其他"等）
- 提供更精准的已分类用户数据分析
- 适用于希望专注分析已整理关注列表的用户

#### 建议清理原则
清理建议功能遵循以下原则和策略：

**清理建议维度**
1. **活跃度维度**
   - 长期不更新内容的UP主（超过6个月无新作品）
   - 活跃度评分低于阈值的用户
   - 最近互动频率极低的关注

2. **内容质量维度**
   - 内容与用户兴趣严重不符的UP主
   - 疑似营销号或垃圾账号
   - 重复内容创作者（同质化严重）

3. **关注价值维度**
   - 粉丝数量与内容质量不匹配的账号
   - 认证状态变化（如失去认证）的UP主
   - 频繁改名或改变内容方向的创作者

**清理策略**
- **温和清理**: 优先建议清理明显无价值的关注
- **精准清理**: 根据用户兴趣偏好个性化推荐
- **批量清理**: 支持按类别或条件批量处理
- **保护重要关注**: 自动识别并保护高价值关注对象

**安全保障**
- 提供详细的清理理由说明
- 支持清理预览和确认机制
- 保留清理历史和恢复功能
- 避免误删重要关注对象
